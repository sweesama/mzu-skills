# ReadGZH

> WeChat Public Account Article Reader for AI Agents

ReadGZH is a high-performance tool that bypasses WeChat's anti-scraping mechanisms with 99.89% success rate. Content is converted to clean, structured Markdown - saving 50-87% token costs.

## Usage

Tell your AI assistant: "Read this article: [WeChat article URL]"

## API

- **MCP endpoint**: `POST https://api.readgzh.site/mcp-server`
- **REST**: `GET https://api.readgzh.site/rd?url=...&format=text`
